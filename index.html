  • uOttawa Schedule Generator


    Side project ~~ Google Calendar API - Node.js
    GitHub repo
    June 2016 - August 2016

    Our courses schedules are provided on the uOttawa portal, but we need to manually put all the courses on Google Calendar if we want to access them easily. Being tired after a few semesters of going through this hassle, I decided to automate the whole process through a webapp.

    The webapp, given the student crendentials, scrapes the calendar and prompts the student to choose the colours for different classes (lectures vs labs vs discussion groups); it then adds the courses to Google Calendar.

  • Mobile Robot


    Physics project ~~ Arduino - Electronics
    GitHub repo
    February 2014 - April 2014

    Part of my program in Cégep was to work on a science project during our last semester, which I decided to do in the physics department. With the help of a teammate, both having no background in electronics (and me having basic programming knowledge in JavaScript), we decided to build an autonomous mobile robot using an Arduino.

    My teammate focused more on the circuity, while I worked on the programming. The robot consisted of 2 independently controlled DC motors, an Arduino, a distance sensor, a collision sensor, a RGB LED, a dual H-Bridge, and an alimentation of 7.5V (for the Arduino) and of 12V (for the motors). The program was constantly reading data from the sensors, doing basic filtering of the data acquired and then outputting a specific voltage to the motors and triggering a specific colour of the RGB LED.

    It was a very inspiring experience, convincing me to go in Software Engineering. Not only that, but our project was rewarded with the best science project presentation of the day (here's the poster, in French only!)

    If you're curious to learn more about the robot, the GitHub repo contains the code used, a brief presentation and also videos of the robot and of the collision sensor we used. The videos are in French, but I put up a description of them here!
